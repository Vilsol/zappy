# Zappy

```
zappy proxies an http server and serves a cached whilst updating cache in the background

Usage:
  zappy [command]

Available Commands:
  help        Help about any command
  proxy       Proxy the provided urls to the ports

Flags:
      --colors       Force output with colors
  -h, --help         help for zappy
      --log string   The log level to output (default "info")

Use "zappy [command] --help" for more information about a command.
```
